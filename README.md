HERO.IO is a modern and interactive web application designed to showcase and manage productive applications. ## ğŸš€ Getting Started

### Prerequisites

Before running this project, make sure you have the following software installed on your computer:

- **Node.js** (v18 or higher)
- **npm** (comes with Node.js) or **yarn**

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nayem7890/hero-apps.git
   cd hero-apps
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   or
   ```bash
   yarn dev
   ```

4. **Open in browser**
   - Navigate to the URL shown in the console (typically `http://localhost:5173`)

### Available Scripts

- **Create production build**
  ```bash
  npm run build
  ```

- **Preview production build**
  ```bash
  npm run preview
  ```

- **Run ESLint**
  ```bash
  npm run lint
  ```
